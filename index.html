사용 팁 및 주의사항

· 지도: VWorld(국토교통부 공간정보 오픈플랫폼) 베이스 + 지하철 노선 오버레이. 주소 지오코딩은 한국 도로명·지번 주소를 우선 시도하고, 안 되면 OpenStreetMap로 폴백합니다. 검색 실패 시 좌표(위도,경도) 직접 입력 또는 매물 추가 후 핀 드래그로 위치 미세조정하세요.

· 🚇🚗🚶 경로 그리기: 매물 상세 모달의 "🚇 대중교통 / 🚗 자가용 / 🚶 도보" 버튼을 누르면 본사→매물 실제 경로가 지도에 색깔 선으로 그려집니다. 소요시간 자동 저장.
  - 대중교통: ODsay(lab.odsay.com, 무료 키 발급, 도메인 등록 필요).
  - 자가용: 카카오모빌리티(developers.kakao.com REST API 키, 일 10,000건 무료, 실시간 교통·통행료·예상 택시비 반영).
  - 도보: OSRM(OpenStreetMap 기반, 무료, 키 불필요).
  - file://에서는 안 되고, https 호스팅(GitHub Pages 등)에서만 작동.

· API 키 변경/초기화: 브라우저 콘솔(F12)에서 localStorage.removeItem('odsay_key') 또는 localStorage.removeItem('kakao_rest_key') 실행 후 새로고침.

· 사진은 자동으로 800px로 압축되어 브라우저에 저장됩니다. 네이버부동산 등에서는 직접 다운로드 후 업로드해야 합니다 (CORS 차단).

· 평당월세는 월세 ÷ 전용평수로 자동 계산됩니다.

· PDF 보고서는 표지 + 요약 비교표 + 매물별 상세카드 3단 구조입니다. PDF 보고서 버튼 → 인쇄 다이얼로그에서 "PDF로 저장" 선택.

· 다른 PC로 이동: 📤 다른 PC로 (HTML) 버튼이 가장 간단 — 데이터가 박힌 단일 HTML 파일을 다운로드 받습니다. 이메일·메신저로 전송 후 받는 PC에서 더블클릭만 하면 자동으로 데이터가 들어옵니다 (복원 단계 없음). 작은 데이터셋만 옮길 땐 JSON 백업도 가능.

· 본사 거리는 직선거리(Haversine)이며 실제 이동거리·시간이 아닙니다.
